AUTELGimbalRotationModeSpeed

Rotate the gimbal’s pitch, roll, and yaw in SPEED Mode. The direction can either be set to clockwise or counter-clockwise. For Phantom 3 Professional, Phantom 3 Advanced and Phantom 3 Standard, roll and yaw rotations are not available. For Inspire 1, Inspire Pro and M100, pitch, roll and yaw rotations are available. For Osmo, roll rotation is not available. The yaw angleVelocity of DJIGimbalSpeedRotation range is (0, 120).
