AUTELGimbalConfig Class Reference

Inherits from NSObject
Declared in AUTELGimbal.h

Overview

This class provides advanced configuration of the gimbal.

  configType

Gimbal configuration type. There are three preset types: fast, medium and slow. When the configuration type to AUTELGimbalUserConfigTypeCustom, user can change the configuration by calling methods related to smoothTracking.

@property (nonatomic, readonly) AUTELGimbalUserConfigType configType

Discussion

Gimbal configuration type. There are three preset types: fast, medium and slow. When the configuration type to AUTELGimbalUserConfigTypeCustom, user can change the configuration by calling methods related to smoothTracking.
