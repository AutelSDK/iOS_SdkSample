AUTELCameraWorkMode

Define camera working mode.

定义相机工作模式，可选模式包括Record、Single、Burst、Timelapse、AEB、Panorama、DelayShot、MotionDelayShot、SlowMotionRecord和LoopRecord、HDR、MFNR
